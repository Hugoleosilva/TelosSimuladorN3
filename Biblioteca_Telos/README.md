## Simulador Télos Nível 3

### Resumo do Projeto:

### Objetivo: 
- Desenvolver um Sistema de Gerenciamento de uma Biblioteca Fictícia.

### Nome da Biblioteca Fictícia: 
- Biblioteca_Telos

## Etapas do Projeto da Biblioteca Télos:

### Criação:

1. Banco de dados: Biblioteca_Telos
2. Tabela Books (Livros)
3. Tabela Users (Usuários)
4. Tabela Loans (Empréstimos)

### Pesquisar e Inserir os Livros e demais dados fictícios:

5. Inserir livros
6. Inserir usuários
7. Inserir empréstimos (alguns devolvidos, outros não)

### Consultas nas tabelas:

8. Buscar livros por gênero
9. Buscar todos os livros emprestados (não devolvidos)
10. Usuários com mais de 3 empréstimos (usando subquery)

### Usando e testando as funções pedidas:

11. Calcular número total de empréstimos por usuário (Função)
12. Livros emprestados e devolvidos (histórico)
13. Usuários com mais empréstimos (ranking)

### Usando procedures pedidas:

14. Registrar empréstimo (com verificação de disponibilidade)
15. Registrar devolução

### Teste das funcionalidades

- Registrar novo empréstimo
- Tentar emprestar livro já emprestado
- Registrar devolução
- Verificar disponibilidade de um livro específico

### Views para relatórios frequentes:

16. View: Livros atualmente emprestados
17. View: Estatísticas de empréstimos por gênero

### Funcionalidades Implementadas:

✅ Modelagem de dados com 3 tabelas relacionadas

✅ Constraints (PK, FK, UNIQUE, NOT NULL)

✅ Inserção, atualização, exclusão e consulta de dados

✅ Joins entre múltiplas tabelas

✅ Subqueries para consultas complexas

✅ Funções armazenadas

✅ Procedures para operações de negócio

✅ Views para relatórios

✅ Verificação de disponibilidade de livros

✅ Relatórios diversos
